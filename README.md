# Causal Information Gravity – Numerical Simulation

This repository contains a Python implementation of the information flux observable \(\Phi(v)\) on a causal set generated by Poisson sprinkling into 2D Minkowski spacetime.

## Description

The observable is defined as:
\[
\Phi(v) = \frac{1}{|D(v,\epsilon)|}
\]
where \(D(v,\epsilon)\) is the causal diamond of half-height \(\epsilon\) centered at \(v\). The code:
- Sprinkles points uniformly in a square \([0,L]\times[0,L]\).
- Counts points in the causal diamond around each vertex.
- Computes the mean and variance of \(\Phi(v)\) over multiple realizations.
- Shows that \(\langle \Phi \rangle \propto 1/\rho\) and \(\operatorname{var}(\Phi) \cdot \rho^2 \to \text{constant}\) as \(\rho \to \infty\).
